# Game Wishlist Manager

A web application for searching video games and managing a personal wishlist.

Student: Ben Danuser
Course: Client-Side Scripting  
Date: 10/8/2025

## Features
- Search for games using API
- Add games to wishlist
- Remove games from wishlist
- Dark/light mode toggle
- Data saves automatically

## Technologies 
- HTML
- CSS
- JavaScript
- API
- Local Storage

## Setup

Open `index.html` in a web browser to run the project.

## How to Use

1. Type a game name in the search box
2. Click "Search" or press Enter
3. Click "Add to Wishlist" on any game
4. Click "My Wishlist" to see saved games
5. Use the theme toggle to switch between light/dark mode

## Course Concepts Used
- DOM manipulation
- Functions and event listeners
- Arrays and objects
- API integration with fetch()
- Local storage
- Async/await

## Challenges Solved
- Preventing duplicate games in wishlist
- Saving data so it doesn't disappear on refresh
- Switching between light and dark themes
- Making the layout responsive for different screen sizes
