# Game Wishlist Manager

A web application for searching video games and managing a personal wishlist with filters, sorting, and theme customization.

**Student:** Ben Danuser  
**Course:** Client-Side Scripting - Final Project  
**Date:** 12/6/2025

**Live Site:** https://bendanuser9.github.io/game-wishlist-manager/  
**GitHub Repository:** https://github.com/bendanuser9/game-wishlist-manager

---

## Features

- Search for games using RAWG API
- Filter by genre and platform
- Sort by name, rating, release date, or popularity
- Add/remove games from wishlist
- Wishlist saves full game data with images
- Dark/light mode toggle
- Data persists automatically

---

## Technologies

- HTML5, CSS3, JavaScript
- RAWG Video Games Database API
- Local Storage API
- Git/GitHub for version control
- GitHub Pages for deployment

---

## How to Use

1. Search for games by name
2. Apply filters (genre, platform) and sorting
3. Click "Add to Wishlist" to save games
4. View saved games in "My Wishlist" tab
5. Toggle theme with the theme button
